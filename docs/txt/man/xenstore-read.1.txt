NAME
    xenstore-read - read Xenstore values

SYNOPSIS
    xenstore-read [*OPTION*]... [*PATH*]...

DESCRIPTION
    Read values of one or more Xenstore *PATH*s.

    -p  Prefix value with key name.

    -s  Connect to the Xenstore daemon using a local socket only.

    -R  Read raw value, skip escaping non-printable characters (\x..).
